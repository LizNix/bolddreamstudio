Featured Work

Bootscores WordPress Starter Theme

bootscores.liznix.com

As a web developer managing a team of other web developers I recognized a need for a WordPress starter theme integrating bootstrap that still lets a coder code. The main advantages of Bootscores are:

  1. It's built on underscores so sane defaults are set for standard WordPress features and nothing to get in the way of development.
  2. It loads the bootstrap framework from the bootstrap CDN. So you have access to all bootstrap’s CSS and JavaScript components without having to serve them on your hosting environment.
  3. It integrates Justin Weaver's Flexnav so multiple level drop downs are easy to add and hover drop down on desktop works out of the box.

Painted Danes

Professional Development Strategy

This industry moves fast and no matter where you are on the path there is always more to learn. New and better web technologies are always emerging, and they are used most effectively by those who put the work into developing core strength, so I prioritize keeping current while constantly deepening foundational knowledge. The following is my personal road map to get myself to the next level.

Currently Learning

JavaScript - bringing javascript competency up to advanced level through online courses and hands on experience in Javascript, followed by the frameworks AngularJS and ReactJS.

PHP - bringing PHP competency to advanced level with online courses and continued hands-on experience.

Accessibility - Learning how mark up is interpreted by accessibility technology and using this very website as a testing ground to make sure my code is compliant and easy to parse for all users. Plans for improvement include completing Google's accessibility course and using online tools to test for compatibility with assistive technologies.

SysAdmin - Accelerating my systems administration learning by hosting my own sites on Digital Ocean and managing the servers there.

Next Steps
